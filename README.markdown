AWS Toolbox
===========

A distribution of AWS CLI tools.

Includes:

  - EC2 API tools
  - EC2 AMI tools
  - CloudWatch tools
  - Auto Scaling tools
  - ELB tools
  - RDS tools
  - Elastic MapReduce tools
  - Elastic Beanstalk tools
  - IAM tools

Installation
------------

Download and run install.sh.
